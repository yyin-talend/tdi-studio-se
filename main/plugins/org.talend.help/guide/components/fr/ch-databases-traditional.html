Databases - Composants traditionnels

Le chapitre Databases - Composants traditionnels décrit les connecteurs de bases de données traditionnelles ainsi que les connecteurs aux bases de données les plus populaires. Ces connecteurs couvrent divers besoins : ouverture de connexion, lecture et écriture de tables, commit de transactions, ainsi que des rollbacks pour le traitement des erreurs. Plus de 40 SGBDR sont supportés. Ces composants sont présents dans la famille Databases de la Palette dans la perspective Integration du Studio Talend.

D'autres types de connecteurs de bases de données, tels que les connecteurs d'appliance, de datawarehouse et de gestion de bases de données, sont documentés dans le Chapitre 8, Databases - Composants d'appliance et de datawarehouse et le Chapitre 9, Databases - Autres composants.

Pour obtenir des informations concernant le support des versions, consultez l'annexe des versions des bases de données/des systèmes tiers supporté(e)s, dans votre Guide d’installation et de migration Talend.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
