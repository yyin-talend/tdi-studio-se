Composants Custom Code

Ce chapitre passe en revue les principaux composants présents dans la famille Custom Code de la Palette dans la perspective Integration du Studio Talend.

La famille Custom Code regroupe tous les composants répondant à un besoin spécifique de création de code à la volée.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
