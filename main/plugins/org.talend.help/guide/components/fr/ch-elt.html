Composants ELT

Ce chapitre passe en revue les propriétés des principaux composants présents dans la famille ELT de la Palette dans la perspective Integration du Studio Talend.

La famille ELT regroupe les connecteurs de bases de données les plus courants, ainsi que des composants de traitement, tous dédiés au mode ELT dans lequel les SGBD cibles deviennent des outils de transformation.

Ce mode supporte les bases de données les plus courantes, comme Teradata, Oracle, Vertica, Netezza, Sybase, etc.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
