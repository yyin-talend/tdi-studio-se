tPostjob

Propriétés du tPostjob

Famille du composant

Orchestration

 

Fonction

Le composant tPostjob démarre l’exécution d’un post-job.

Objectif

Le tPostjob déclenche une action requise après l’exécution d’un Job.

Utilisation

Le tPostjob est un composant de début. Il ne peut être relié au composant suivant qu’avec une connexion de type Iterate.

Connections

 

Liens de sortie (d’un composant à un autre) :

Trigger : OnComponentOk.

Liens d’entrée (d’un composant à un autre) :

Trigger : Synchronize, Parallelize.

Pour plus d’informations concernant les liens, consultez le Guide utilisateur de Talend Open Studio.

Pour plus d’informations sur le composant tPostjob, consultez le Guide utilisateur de Talend Open Studio.

Scénario associé

Aucun scénario n’est disponible pour ce composant.
