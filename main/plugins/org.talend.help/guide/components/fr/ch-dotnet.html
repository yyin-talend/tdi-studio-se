Composants DotNET

Ce chapitre contient les composants que vous pouvez trouver dans la famille DotNET de la Palette dans la perspective Integration du Studio Talend.

La famille DotNET comprend les connecteurs les plus fréquemment utilisés pour communiquer avec des objets .NET.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
