tJBossESBOutput

Propriétés du tJBossESBOutput

Famille de composant

Internet

 

Fonction

Le composant tJBossESBOutput transforme les données utilisées dans un Job Talend en un message JBossESB.

Objectif

Le composant tJBossESBOutput permet de passer les données d’un Job Talend vers un message JBossESB.

Basic settings

Message key

Saisissez la clé du message à écrire dans le serveur JBossESB.

 

Message value

Saisissez le flux de données qui sera écrit comme contenu du message dans le serveur JBossESB.

Advanced settings

tStatCatcher Statistics

Cochez cette case pour collecter les données de log au niveau du Job ainsi qu’au niveau de chaque composant.

Utilisation

Ce composant doit être lié à un composant d’entrée ou intermédiaire via un lien de type Trigger.

Scénario associé

Aucun scénario n’est disponible pour ce composant.
