tPetalsOutput

Note

Pour plus d’informations concernant l’ESB et l’intégration de Petals dans Talend, consultez le Guide utilisateur de Talend Data Integration Studio.

Propriétés du tPetalsOutput

Famille de composant

Internet

 

Fonction

Le tPetalsOutput transforme un flux de données Petals en un flux pouvant être utilisé dans un Job Talend.

Objectif

Le tPetalsOutput permet de passer des données d’un Job Talend à l’ESB Petals.

Basic settings

Schema et Edit Schema

Un schéma est une description de lignes, il définit le nombre de champs qui sont traités et passés au composant suivant. Le schéma est soit local (built-in) soit distant dans le Repository.

Cliquez sur Edit Schema pour modifier le schéma. Notez que si vous modifiez le schéma, il devient automatiquement built-in.

Advanced settings

tStatCatcher Statistics

Cochez cette case pour collecter les données de log au niveau du composant.

Utilisation

Ce composant requiert un composant intermédiaire ou de début.

Limitation

Un Job contient plusieurs instances de tPetalsOutput, mais ces composants de sortie doivent avoir le même schéma.

Scénario associé

Aucun scénario n’est disponible pour ce composant.
