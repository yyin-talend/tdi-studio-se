Databases - appliance/datawarehouse components

This chapter describes connectors for specific databases oriented to the processing of large volume of data. These connectors cover various needs, including: opening connections, reading and writing tables, committing transactions as a whole, and performing rollback for error handling. These components can be found in the Palette of the Integration perspective of the Talend Studio.

Other types of database connectors, such as connectors for traditional databases and database management, are documented in Chapter 7, Databases - traditional components and Chapter 9, Databases - other components.

For database version support information, see the appendix on supported database versions of the Talend Installation and Upgrade Guide of the Talend solution you are using.

Warning

For Talend Open Studio for Big Data, the Property type, Schema and Query Type of components are always Built-in. For how to edit a Built-in schema, see Talend Data Integration Studio User Guide.
