Composants Technical

Ce chapitre fournit des informations détaillées concernant les composants de la famille Technical de la Palette dans la perspective Integration du Studio Talend.

Les composants Technical sont des composants orientés Java et effectuent des actions très techniques, comme le chargement de données dans la mémoire (en de petits sous-ensembles d'informations) et leur conservation pour un usage ultérieur lors du traitement.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
