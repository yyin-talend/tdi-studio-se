Composants Misc

Ce chapitre passe en revue les principaux composants présents dans la famille Misc de la Palette dans la perspective Integration du Studio Talend.

La famille Misc regroupe divers connecteurs permettant de répondre à des besoins, tels que la création de lignes de données factices, la mise en tampon de données, le chargement de variables de contexte.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
