tFSTransform

Propriétés du tFSTransform

Famille de composant
FileScale
Fonction
Le composant tFSTransform vous permet d'effectuer des changements spécifiques sur des colonnes spécifiques lors du traitement du fichier. Ce composant permet de transformer les colonnes des fichiers volumineux en temps réel. Pour optimiser les performances, le composant trie généralement les données avant de les traiter.
Objectif
Le FSTransform permet de transformer les colonnes de fichiers volumineux.
Basic settings
Schema type et Edit Schema
Un schéma est une descritption de lignes, i.e., il définit le nombre de champs qui sont traités et passés au composant suivant. Le schéma est soit local (built-in) soit distant dans le Repository.
Cliquez sur Edit Schema pour modifier le schéma.

Si vous modifiez le schéma, il se met automatiquement en built-in.

 
 
Repository : Le schéma existe déjà et est stocké dans le Repository. Ainsi, il peut être réutilisé dans divers projets et job designs. Voir également : Paramétrer un schéma du Repository du Guide Utilisateur de Talend Integration Suite.
 
 
Built-in : Le schéma est créé et conservé pour ce composant uniquement. Voir également : Paramétrer un schéma built-in du Guide Utilisateur de Talend Integration Suite.
 
Property type
Peut être Built-in ou Repository.
 
 
Built-in : Propriétés utilisées ponctuellement.
 
 
Repository : Sélectionnez le fichier de propriétés du composant. Les champs suivants sont alors pré-remplis à l'aide des données collectées.
 
Input File Name
Nom du fichier contenant les données que vous souhaitez filtrer.
 
Output File Name
Nom du fichier dans lequel vous souhaitez écrire les données filtrées.
 
Record separator (char)
Caractère, chaîne ou expression régulière séparant les lignes.
 
Field separator (char)
Caractère, chaîne ou expression régulière séparant les champs.
 
Header
Nombre de lignes d'en-tête à ignorer en début de fichier.
 
Footer
Nombre de lignes à ignorer en fin de fichier.
 
Transform Code
Saisissez le code FSLang à utiliser pour transformer la colonne spécifiée.
Utilisation
Ce composant traite des fichiers, ainsi il ne nécessite pas de flux de données en entrée ou en sortie. Il sert à vérifier les données contenues dans des fichiers volumineux.
Limitation
La limitation dépend des limites de la mémoire physique et de l'architecture du processeur. Par exemple, la taille totale des fichiers traités ne peut pas excéder la limite du système de fichiers pour supporter les fichiers très volumineux (maximum de la valeur 64 bits signée).
