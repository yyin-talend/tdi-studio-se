tMomConnection

Propriétés du tMomConnection

Ce composant est étroitement lié aux composants MomCommit et tMomRollback. Ils sont généralement utilisés avec un composant tMomConnection car il permet d’ouvrir une connexion pour la transaction en cours.

Famille de composant

Internet

 

Fonction

Le composant tMomConnection ouvre une connexion à un serveur MQ.

Objectif

Ce composant ouvre une connexion vers un serveur MQ dans un but de communication.

Basic settings

Failover

Cochez cette case pour afficher la table d'adresses (hôte et port) du serveur.

Dans cette table, vous pouvez définir différents serveurs MQ pour le failover.

 

MQ server

Liste des serveurs MQ.

 

Host

Adresse IP du serveur MQ.

 

Port

Port d'écoute du serveur MQ.

Advanced settings

Use Transacted

Cochez cette case pour configurer la session pour laquelle effectuer une transaction.

 

tStatCatcher Statistics

Cochez cette case pour collecter les données de log, aussi bien au niveau du Job qu’au niveau de chaque composant.

Utilisation

Ce composant est généralement utilisé avec des composants Mom, notamment les composants tMomCommit et tMomRollback.

Scénario associé

Pour un scénario associé au tMomConnection, consultez la la section intitulée « tMysqlConnection ».
